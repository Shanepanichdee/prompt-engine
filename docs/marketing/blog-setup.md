# Blog Setup Notes

## Platform: Hashnode

**Steps to set up:**
1. Go to hashnode.com → Sign up with GitHub or Google → Create a new blog
2. Publication name: `Prompt Engine Blog`
3. Handle: `prompt-engine` (or `prompt-engine-blog` if taken)
4. Optional: Set custom domain `blog.prompts.data-shane.com`
   - Hashnode settings → Domains → add `blog.prompts.data-shane.com`
   - Cloudflare DNS → CNAME: `blog` → `hashnode.network`
