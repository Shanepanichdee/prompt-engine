# Prompt Engine

Rule-based multilingual prompt engineering system. Assemble structured prompts from 14 frameworks with multilingual connector support.

## Packages

- `packages/core` — pure TypeScript library, zero deps, npm-publishable
- `apps/web` — Next.js App Router UI (client-side only)
- `packages/cli` — interactive CLI via `npx prompt-engine`

## Frameworks

CRISPE · CO-STAR · RISEN · RTF · Chain of Thought · Few-Shot · APE · ReAct · BROKE · RODES · TRACE · CARE · Structured Output · Auto-CoT

## Languages

English · Thai · Chinese · Japanese · Korean · Spanish · French · German · Portuguese · Arabic

## Getting Started

```bash
pnpm install
pnpm dev
```

## CLI

```bash
npx prompt-engine
npx prompt-engine --lang th --json
```
