# Prompt Engine Web

Next.js App Router UI for prompt building.

## Run

```bash
pnpm --filter web dev
```

Open `http://localhost:3000`.

## Features

- Two-panel builder/output layout
- 14 framework picker with details and samples
- 10 language selector (native labels)
- Live prompt generation from `@prompt-engine/core`
- Copy prompt, Dev Mode, Compact Mode
- Branding header with logo + credit
